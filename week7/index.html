    Chapter 11

    Function Properties and Methods

    Functions are first-class objects – the can be passed around like all other values

    Functions have properties and Methods themselves

    Ex function.length will return the number of parameters it has

    Call and Apply Methods

    Call() method sets this inside a function – the first argument passed

    Ex. sayHello.call(clark);

    Call() method takes the first parameter, whereas the actual function accepts the parameters after the first

    Custom Properties

    Can add properties to functions like any other object

    Ex. Square.description = “” Can set a description attribute to a function called Square

    Memoization

    Can save results in a cache property

    IIFE

    Can create Temporary Variables

    Can run Initialization code

    Can use strict mode safely

    Function’s that redefine themselves

    A function can redefine itself by calling it’s name and then assigning it a new function

    Recursive Functions

    Call a function within itself with some kind of flag to return if met

    Functions can return functions

    Use these when a parameter can change the function

    Generator

    Returns a generator object – essentially an easy way to create objects that use other complex entities to be a part of their attributes

    Chapter 13 - Ajax

    Redirect

    Redirect() method can be used to redirect to another URL – creates a new promise that resolves to the response from the redirected URL

    Blob

    The blob() method is used to read a file of raw data, such as an image or a spreadsheet

    Headers Interface

    Fetch API introduced a Headers interface, which can be used to create a Headers object.

    An extremely important miscellaneous piece of information is that AJAX calls are async – so make sure to handle code and promises accordingly
